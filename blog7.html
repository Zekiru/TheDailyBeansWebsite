Software Development in the Industry

A Blog Post by Ezekiel Villasurda, October 12 2022.

Software, in definition are sets of instructions for a computers to follow, Developers are responsible for making these Softwares, hence they are known as Software Developers, the people who are responsible for making all digital computers function, like the website you are viewing right now.

In the Industry, Software Developers continue to grow in demand, describing how important they are for technological companies to operate and make great ideas come to life.

As Software Developers, they can show and make use of their knowledge in coding, logical thinking, and most importantly their creativity and innovative thinking, which is fitting for the 9th the Sustainable Development Goal of Building resilient infrastructures, promote inclusive and sustainable industrialization and foster innovation.

Not saying one should have coding talent to take this career, all of these skills can be learned with time, but if you are interested in things related to Computing, Software Applications, Websites, or even Game Development, which is a popular choice for young developers, it would not hurt to try this track/course.

If you want some advice on how to start getting into Software Development, learning how to make Websites is a good start, it is basic compared to Developing other Softwares like Games or Applications, it also has practical uses outside of the Software Development Strand, since the Web is already a part of our daily lives.

Author's Note:
Not only did I write this Blog, I made this whole solely by coding, which means everything you see here comes from multiple lines of code manually typed in, it is how I learned how websites work (Not that Web Development Applications were not an option, it was worth it in the end).

If I can learn how to code on my own, then so can you!
